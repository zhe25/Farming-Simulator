To import project to eclipse:
- Open a new or existing workspace.
- Select File>Import.
- General>Existing Projects Into Workspace.
- Select Archive File>Browse>Exported Project.zip
- Ensure Farming-Simulator is selected under projects.
- Select Finish.
- To run the program, run from the main.swing.GameWindow class.
- To run in command line mode, add "text" as an argument to the run configuration.

To run the final Jar file:
- Ensure Java 8 or higher is installed.
- Open a terminal in the same directory as FarmingSimulator.jar
- To run in window mode, run: "java -jar FarmingSimulator.jar"
- To run in command line mode, run with argument text: "java -jar FarmingSimulator.jar text"
